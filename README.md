# ToDoList
## Installation

```bash
$ npm install express --save
```

## Usage
### Linux
 - we have Makefile

```bash
# start docker-compose
$ make start 

# stop docker-compose
$ make stop 
```
### Windows
```bash
# start docker-compose
$ docker-compose up -d 

# stop docker-compose
$ docker-compose down
```


## Test

```bash
# unit tests
$ npm test

# tests for each file
$ npm test {filename ex: user}
```
